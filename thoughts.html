Fake APIs for FrontEnd-Development

I just read an article on dev.to about Fake APIs for FrontEnd Development. The community came up with some good training APIs for gathering data. I'll list some of them here in no particular order:

JSONPlaceHolder provides an API with GET, POST, PUT and DELETE requests and supports CORS. The site has an endpoint with preloaded data

REQ | RES Similar to JSONPlaceHolder, provides an endpoint for all HTTP actionswith preloaded data

Any-API provides documentation and test consoles for over 1400 public APIs

Open Weather API The mother of all Weather APIs, used in several internet tutorials...

CRUDpi.ioprovides a standalone CRUD API with some preloaded data
